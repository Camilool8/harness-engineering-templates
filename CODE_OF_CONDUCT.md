# Code of Conduct

This project adopts the **[Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)** as its code of conduct. Contributors, maintainers, and anyone interacting with this project's spaces — issues, pull requests, discussions — agree to abide by it.

The canonical text (Our Pledge, Our Standards, Enforcement Responsibilities, Scope, Enforcement, Enforcement Guidelines, Attribution) lives at:

<https://www.contributor-covenant.org/version/2/1/code_of_conduct/>

A plain-text copy is also available at:

<https://www.contributor-covenant.org/version/2/1/code_of_conduct/code_of_conduct.txt>

## Reporting

Report conduct concerns by opening a [private security advisory](https://github.com/Camilool8/harness-engineering-templates/security/advisories/new) or contacting the repo owner [@Camilool8](https://github.com/Camilool8) directly.

All reports are handled confidentially. The maintainer will review and respond as promptly as possible, and will follow the [Enforcement Guidelines](https://www.contributor-covenant.org/version/2/1/code_of_conduct/#enforcement-guidelines) section of the Contributor Covenant when deciding consequences.
